# Easifier: GOL IBE

Source: [CEE Travel Systems - GOL IBE](https://www.cee-systems.com/solutions/gol-ibe)

## Short Client Summary

**GOL IBE** is an online booking engine for travel agencies that lets them sell air tickets directly on their own website. It covers search, booking, payment, and ticketing, while connecting multiple content sources in one tool.

It is suitable for agencies that want to:

- sell flights through their own website,
- offer broader content than standard GDS only,
- control pricing, fees, and ticketing in one place,
- start with a basic setup and later move to a more advanced enterprise model.

## What the Product Does

### 1. Content Sources

- **GDS content**: standard airline content from global distribution systems.
- **Low-cost carriers**: access to low-cost airline content.
- **NDC content**: modern airline-direct content connections.
- **Rail**: the option to include rail content as part of the offer.

In practice, this means a broader range of travel options in one booking flow.

### 2. Search and Flight Selection

- **One-way / Return / Multi-city**: supports basic and more complex trip types.
- **Passenger type search**: allows predefined passenger category logic.
- **One-way combinations**: combines one-way flights into better itinerary options.
- **Branded fares**: shows fare bundles, conditions, and baggage details.
- **Parallel search**: runs simultaneous searches for broader result coverage.
- **MultiPCC**: searches across multiple PCCs / markets, useful for international coverage.

### 3. Branding and Customization

- **Color scheme / CSS changes**: adapts the tool to the client brand.
- **Custom domain**: runs the booking tool on the agency's own domain.
- **Custom notifications and attachments**: adjusts communication settings to agency needs.

### 4. Sales and Checkout

- clear search results with timings, durations, direct-flight indicators, conditions, and baggage,
- **promo codes** for discounts and campaigns,
- **payment via bank gateways** or booking without immediate payment,
- **seat booking**,
- **multiple flights / OpenJaw** support for more advanced itineraries.

### 5. Ticketing and Operations

- **automatic ticketing**: tickets can be issued automatically,
- **manual ticketing**: tickets can also be issued from the admin console,
- **24/7 admin console**,
- settings for **service fees**, **airline commissions**, and discounts,
- management of registered customers and sub-agents,
- **dealer sales** support for partner distribution.

### 6. Additional Tools

- **Flight Watchdog**: monitoring selected flights or pricing situations via watchers.
- **Google Analytics + Tag Manager**: analytics and marketing measurement.
- **Statistics and data export**: reporting and export functions.
- **CEE Marketplace**: access to additional travel-related services.
- **Meta-search integration**: distribution into metasearch channels.
- **Offline bookings retrieve**: importing offline bookings into the system.

## Product Versions

## Standard

Pricing:

- **$0 / month**
- **$1.67** per ticketed Travelport+ booking
- **$3.60** per Travelfusion booking
- **$390 / year** non-refundable deposit

Best for:

An agency that wants to launch online flight sales quickly with no monthly platform fee.

Typically includes:

- core search and booking functionality,
- Travelport+ and related air content,
- admin console,
- branded fares,
- multiple-flight / OpenJaw support,
- seat booking,
- basic design customization through the admin console,
- core language support,
- 1 admin training session included,
- no included Flight Watchdog watchers.

Note:

The public pricing table is visually compressed, so the Standard tier should be treated as the base package rather than a perfectly detailed feature-by-feature matrix.

Best for:

An agency that wants more control over pricing, marketing, and partner sales.

Compared with Standard, it appears to add:

- more advanced fee and pricing controls,
- **Google Analytics + Tag Manager**,

Larger or multi-market agencies that need the fullest setup and higher operating limits.

## Plain-English Explanation of Key Features

- **Travelport+ booking fee**: fee charged for each ticketed booking from Travelport content.
- **Travelfusion booking fee**: fee charged for low-cost bookings via Travelfusion.
- **Deposit**: prepaid annual credit from which booking fees are deducted.
- **Flight Watchdog**: automated monitoring of selected fare or flight situations; watcher count defines capacity.
- **Dealer sales**: enables selling through partners or sub-agents.
- **MultiPCC**: combines content from multiple PCCs / markets to improve coverage and competitiveness.
- **Branded fares**: shows what each fare package includes, such as baggage or extra services.
- **Automated e-ticketing**: issues tickets automatically once conditions are met, usually after payment.
- **Offline bookings retrieve**: brings offline bookings into the online system for admin and reporting.
- **Meta-search integration**: allows the product offer to be distributed into flight comparison / metasearch environments.

## Important Notes and Conditions

- According to the website, monthly fees start once the search form is made public.
- Fees apply to **ticketed bookings**, not just created reservations.
- The deposit is described as **annual and non-refundable**.
- Low-cost content via Travelfusion may require a local payment gateway or separate gateway integration.
- Some features are listed as available on request or depending on implementation.
- **Price display in different currencies** is listed as available only in the previous design version.
